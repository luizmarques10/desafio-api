# desafio-api

## Dependencies

* Maven
* Java 8 (Open SDK)

## Test Execution

### Environments

* uat

### Run all tests

`$ mvn clean test -DargLine="-Dkarate.env=uat"`

## Test Report

* Karate Report: `karate_api/target/surefire-reports/karate-summary.html`
* Cucumber: `karate_api/target/cucumber-html-reports`
